== Description ==

**MRK Mortgage Calculator** is a powerful, modern, and highly customizable mortgage/loan calculator plugin for WordPress.

It allows your visitors to instantly calculate:
- Monthly Interest-Only Payment
- Total Interest-Only Payment
- Monthly Repayment (Principal + Interest)
- Total Repayment Amount

The calculator features real-time sliders, comma formatting, mobile-friendly design, and **complete visual control** from the WordPress admin panel — no coding required.

### Key Features

* Fully responsive & mobile optimized
* Live range sliders synced with input fields
* Supports Interest-Only and Full Repayment calculations
* Beautiful modern design with dynamic colors and typography
* Complete control over text, labels, defaults, ranges, colors, fonts & layout
* Google Fonts support
* Shortcode ready: `[mortgage_calculator]`
* Lightweight and fast (no unnecessary bloat)
* Clean code with proper sanitization and escaping

### Perfect For
- Real estate websites
- Mortgage brokers & financial advisors
- Property developers
- Personal finance blogs
- Banking & loan comparison sites

== Installation ==

1. Upload the plugin folder `mrk-mortgage-calculator` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to **Settings → MRK Mortgage Calc** to configure the calculator
4. Use the shortcode `[mortgage_calculator]` on any page or post

== Frequently Asked Questions ==

= How do I display the calculator? =
Simply add the shortcode `[mortgage_calculator]` to any page or post.

= Can I change the colors and fonts? =
Yes! You have full control over background, text, button, card colors, typography, and layout from the admin settings.

= Does it support Interest-Only and Repayment mortgages? =
Yes, it calculates both Interest-Only and Full Repayment (amortizing) options simultaneously.

= Is it mobile friendly? =
Absolutely. The calculator is fully responsive and optimized for mobile devices (including proper input handling on iOS).

= Can I change the currency symbol? =
Yes, you can set any currency symbol ($, £, €, ₹, etc.) from the settings.
